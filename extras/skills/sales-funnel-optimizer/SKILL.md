# Sales Funnel Optimizer

Otimização de funis de venda automatizados com IA.

## Métricas-Chave

```
Novo Lead → Primeiro Contato:   Meta > 90%
Primeiro Contato → Qualificado: Meta > 60%
Qualificado → Proposta:         Meta > 40%
Proposta → Fechamento:          Meta > 20%
```

## Indicadores de Problema
- Leads travando em um estágio → prompt do subagente precisa ajuste
- Alta taxa de não-resposta → mensagem inicial fraca ou timing ruim
- Objeções recorrentes não tratadas → adicionar handling no prompt
- Mensagens muito longas → cliente perde interesse

## Framework: Diagnosticar → Hipótese → Implementar → Medir

1. **Diagnosticar**: Qual estágio tem maior drop-off?
2. **Hipótese**: "Se mudarmos X, então Y melhora porque Z"
3. **Implementar**: Uma mudança por vez, manter versão anterior
4. **Medir**: Mínimo 50 interações ou 1 semana antes de avaliar

## Padrões de Mensagens WhatsApp

- **Cold**: Referência ao produto + pergunta aberta. Máximo 2-3 linhas.
- **Warm**: Referência à conversa anterior + info de valor + pergunta de avanço.
- **Fechamento**: Resumo + proposta clara + próximos passos.
- **Objeção**: Validar → Reframe → Pergunta de avanço. Nunca argumentar contra.

## Regras
- SEMPRE documente mudanças em PATTERNS.md
- Uma mudança por vez para medir impacto isolado
- Manter versão anterior para rollback
- Se melhorou: documentar como padrão. Se piorou: rollback.
